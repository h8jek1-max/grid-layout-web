# grid-layout-web
A lightweight, high-performance JavaScript engine designed for automatic, collision-free arrangement of UI elements in a 2D grid. Perfect for dashboards, photo books, or masonry-style layouts.

# 2D Geometric Grid Layout Engine

## Features
- **Collision Avoidance:** Smart boundary detection ensures elements never overlap.
* **Pure Vanilla JS:** Zero external libraries (no jQuery, no React). Just high-speed native JavaScript.

## Origin & Credits
This project is a result of collaborative evolution:
- **Concept & Initial Code:** Created by Miroslav Hájek to solve layout issues in automated photo book generation.
- **Optimization & Logic:** Refined with the help of Google Search AI and Claude AI to fix layout stability and grid calculations.
- **Final Architecture:** Polished by Gemini AI into a fully functional, production-ready application.

## Usage
Simply add the `.grid-item` class to your elements within the `#grid-container`. Adjust the `CONFIG` object in the `<script>` tag to customize gaps, min/max dimensions, and aspect ratio behavior.
